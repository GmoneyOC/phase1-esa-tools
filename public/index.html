Data Sources Dashboard

Interactive reference of 29 federal, state, and geologic databases used in California groundwater investigations. Filter by category, data type, API availability, and GIS services.
  • 15 federal databases (USGS, EPA, FEMA, USDA, NOAA)
  • 12 California state databases (GeoTracker, GAMA, EnviroStor, CASGEM)
  • API endpoints and GIS/WMS service URLs
  • Live charts and sortable, searchable table

Report Generator

Step-by-step Phase I ESA report builder. Enter site info, track database searches with auto-dating, fill physical setting data, document RECs, and generate a formatted preview.
  • 3 site types: Commercial, Agricultural, Residential
  • 28-database checklist with search tracking
  • Physical setting fields (soils, flood, seismic, hydro)
  • Print to PDF or copy into .docx templates
Open Generator

Quick Start

Deploy to Vercel

git clone https://github.com/GmoneyOC/phase1-esa-tools.git
cd phase1-esa-tools
npx vercel

Or click "Deploy" on Vercel and point it at the repo. Zero config needed.

Run Locally

git clone https://github.com/GmoneyOC/phase1-esa-tools.git
cd phase1-esa-tools
npx serve public

No build step. All files are static HTML/JS/CSS with no dependencies.
